Gii thiu v mi trng Colab/Jupyter Notebooks

Colab/Jupyter notebooks (sau ny gi tt l notebooks) cho php ngi dng thc hin mt lng ln phn tch d liu v kim nh thng k. Chng ta s trnh by mt vi k thut n gin y.

Code Cells vs. Text Cells

Nh chng ta thy, c 2 loi cell trong notebooks

  • Code Cells: S dng nhp vo dng lnh code thc thi lnh, thng vit bng ngn ng lp trnh Python
  • Text Cells: S dng cho vic nhp ngn ng t nhin, li din gii trong notebooks, s dng cu trc nh dng Markdown

Thc thi mt Lnh

Mt Code Cell s c thc thi khi bn nhn nt chy (run), hoc khi bn nhn t hp phm shift-enter. Vic thc thi mt Code Cell s thc hin tng dng m theo th t v in kt qu ca dng cui cng bn di .

In[]:
2 + 2
Out[]:
4

i khi khng c kt qu no in ra, nh trng hp vi vic gn gi tr

In[]:
X = 2

Hy nh rng ch c kt qu t dng cui cng c in ra

In[]:
2 + 2
3 + 3
Out[]:
6

Tuy nhin, bn c th in bt k dng no bn mun bng cch s dng cu lnh print.

In[]:
print(2 + 2)
3 + 3

4
Out[]:
6

Nhn bit Cell ang chy

Khi mt ang chy, mt [*] (hoc biu tng quay quay) s hin th bn tri. Khi cha c thc hin, s hin th [ ]. Khi n c chy, mt s s hin th ch ra th t m n c thc hin trong qu trnh chy notebooks [5]. Hy th trn ny v ghi ch li iu .

In[]:
# Take some time to run something
c = 0
for i in range(10000000):
    c = c + i
c
Out[]:
49999995000000

Lu rng bn c th i tn cc th vin thnh bt c tn g bn mun sau khi nhp. Cu lnh as cho php iu ny. y, chng ti s dng np v pd lm b danh cho numpy v pandas. y l mt b danh rt ph bin v s c tm thy trong hu ht cc on code trn mng. Mc ch ca iu ny l bn c th g t k t hn khi bn thng xuyn truy cp vo cc th vin ny.
