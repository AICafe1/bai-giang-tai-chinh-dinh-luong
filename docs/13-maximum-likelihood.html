By gi chng ta s nh ngha cc hm m, da trn d liu ca chng ta, s tnh ton MLE cho cc tham s $\mu$ v $\sigma$ ca phn phi chun.

Hy nh li cng thc

$$\hat\mu = \frac{1}{T}\sum_{t=1}^{T} x_t$$

$$\hat\sigma = \sqrt{\frac{1}{T}\sum_{t=1}^{T}{(x_t - \hat\mu)^2}}$$

In[3]:
def normal_mu_MLE(X):
    # Get the number of observations
    T = len(X)
    # Sum the observations
    s = sum(X)
    return 1.0/T * s

def normal_sigma_MLE(X):
    T = len(X)
    # Get the mu MLE
    mu = normal_mu_MLE(X)
    # Sum the square of the differences
    s = sum( np.power((X - mu), 2) )
    # Compute sigma^2
    sigma_squared = 1.0/T * s
    return math.sqrt(sigma_squared)

By gi hy th cc hm ca chng ta trn d liu mu v so snh chng vi hm np.mean v np.std c sn.

numpy nh ngha phn phi m l $$\frac{1}{\lambda}e^{-\frac{x}{\lambda}}$$

V vy, chng ta cn s dng gi tr nghch o ca MLE. Ti , n c th hin l

$$\hat\lambda = \frac{T}{\sum_{t=1}^{T} x_t}$$

y ch l nghch o, v vy

$$\hat\lambda = \frac{\sum_{t=1}^{T} x_t}{T}$$

def exp_lamda_MLE(X):
    T = len(X)
    s = sum(X)
    return s/T

Tt nhin, vic c lng ny s khng c ngha nu chng ta cha kim tra xem chng c tun theo phn phi chun hay khng. Chng ta c th kim tra iu ny bng cch s dng kim nh Jarque-Bera. Kim nh Jarque-Bera s bc b gi thuyt v phn phi chun nu gi tr p-value nh hn mt ngng.
