H s tng quan hng Spearman

Trc gic

Trc gic y l thay v nhn vo mi quan h gia hai bin, chng ta s xem xt mi quan h gia cc hng. iu ny gip trnh nh hng ca cc gi tr ngoi lai v quy m ca d liu.

nh ngha

Tham s method='average' ch ra rng khi c s ng hng, chng ta s tnh trung bnh cc hng m cc s s chim. V d, hai s $5$ trn, s chim cc hng 1 v 2, mi s s c gn mt hng l $1.5$.

tnh h s tng quan hng Spearman cho hai tp d liu $X$ v $Y$, mi tp c kch thc $n$, chng ta s dng cng thc $$r_S = 1 - \frac{6 \sum_{i=1}^n d_i^2}{n(n^2 - 1)}$$

trong $d_i$ l s khc bit gia cc hng ca cp quan st th $i$, $X_i - Y_i$.

Kt qu s lun nm trong khong t $-1$ n $1$. Gi tr dng cho thy mi quan h dng gia cc bin, trong khi gi tr m cho thy mi quan h nghch. Gi tr 0 ng khng c mi quan h n iu no. iu ny khng c ngha l khng c mi quan h; v d, nu $Y$ bng $X$ vi tr 2, chng c mi lin h mt cch n gin v chnh xc, nhng $r_S$ ca chng c th gn bng khng:

Th nghim

Hy xem iu g s xy ra nu chng ta ly $X$ t phn phi Poisson (khng phi phn phi chun), v sau t $Y = e^X + \epsilon$ trong $\epsilon$ c ly t phn phi Poisson khc. Chng ta s tnh ton h s hng Spearman v h s tng quan trn d liu ny, sau thc hin ton b th nghim nhiu ln. Bi v $e^X$ to ra nhiu gi tr rt xa so vi phn cn li, chng ta c th coi y nh l m hnh 'cc im ngoi lai' trong d liu ca chng ta. H s hng Spearman nn cc im ngoi lai v lm tt hn trong vic o lng tng quan. Tng quan bnh thng b nhm ln bi cc im ngoi lai v trung bnh s o lng t hn mi quan h thc s so vi nhng g ang c.

## Let's see an example of this
n = 100

def compare_correlation_and_spearman_rank(n, noise):
    X = np.random.poisson(size=n)
    Y = np.exp(X) + noise * np.random.normal(size=n)

    Xrank = stats.rankdata(X, method='average')
    # n-2 is the second to last element
    Yrank = stats.rankdata(Y, method='average')

    diffs = Xrank - Yrank # order doesn't matter since we'll be squaring these values
    r_s = 1 - 6*sum(diffs*diffs)/(n*(n**2 - 1))
    c_c = np.corrcoef(X, Y)[0,1]

    return r_s, c_c

experiments = 1000
spearman_dist = np.ndarray(experiments)
correlation_dist = np.ndarray(experiments)
for i in range(experiments):
    r_s, c_c = compare_correlation_and_spearman_rank(n, 1.0)
    spearman_dist[i] = r_s
    correlation_dist[i] = c_c

print('Spearman Rank Coefficient: ' + str(np.mean(spearman_dist)))
# Compare to the regular correlation coefficient
print('Correlation coefficient: ' + str(np.mean(correlation_dist)))

n = 100
noises = np.linspace(0, 3, 30)
experiments = 100
spearman = np.ndarray(len(noises))
correlation = np.ndarray(len(noises))

for i in range(len(noises)):
    # Run many experiments for each noise setting
    rank_coef = 0.0
    corr_coef = 0.0
    noise = noises[i]
    for j in range(experiments):
        r_s, c_c = compare_correlation_and_spearman_rank(n, noise)
        rank_coef += r_s
        corr_coef += c_c
    spearman[i] = rank_coef/experiments
    correlation[i] = corr_coef/experiments

plt.scatter(noises, spearman, color='r')
plt.scatter(noises, correlation)
plt.legend(['Spearman Rank', 'Regular Correlation'])
plt.xlabel('Amount of Noise')
plt.ylabel('Average Correlation Coefficient')
