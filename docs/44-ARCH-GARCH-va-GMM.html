M phng Trng hp GARCH(1, 1)

Chng ta s bt u bng cch s dng ly mu Monte Carlo m phng mt qu trnh GARCH(1, 1). Gi tr ng ca chng ta s l

$$\sigma_1 = \sqrt{\frac{a_0}{1-a_1-b_1}} \\ \sigma_t^2 = a_0 + a_1 x_{t-1}^2+b_1 \sigma_{t-1}^2 \\ x_t = \sigma_t \epsilon_t \\ \epsilon \sim \mathcal{N}(0, 1)$$

Cc tham s ca chng ta s l $a_0 = 1$, $a_1=0.1$, v $b_1=0.8$. Chng ta s loi b 10% gi tr m phng u tin (thi gian t chy).

def simulate_GARCH(T, a0, a1, b1, sigma1):

    # Initialize our values
    X = np.ndarray(T)
    sigma = np.ndarray(T)
    sigma[0] = sigma1

    for t in range(1, T):
        # Draw the next x_t
        X[t - 1] = sigma[t - 1] * np.random.normal(0, 1)
        # Draw the next sigma_t
        sigma[t] = math.sqrt(a0 + b1 * sigma[t - 1]**2 + a1 * X[t - 1]**2)

    X[T - 1] = sigma[T - 1] * np.random.normal(0, 1)

    return X, sigma

        A[1, k] = 1 - stats.norm.cdf(k + 1) # Compare to Gaussian distribution
    return A

Kim tra Hnh vi ARCH

Bc u tin l kim tra cc iu kin ARCH. lm iu ny, chng ta thc hin hi quy trn $x_t$ theo m hnh sau.

$$x_t^2 = a_0 + a_1 x_{t-1}^2 + \dots + a_p x_{t-p}^2$$

Chng ta s dng OLS c lng $\hat\theta = (\hat a_0, \hat a_1, \dots, \hat a_p)$ v ma trn hip phng sai $\hat\Omega$. Sau , chng ta c th tnh ton thng k kim tra

$$F = \hat\theta \hat\Omega^{-1} \hat\theta'$$

Chng ta s bc b gi thuyt nu $F$ ln hn cc thanh tin cy 95% trong phn phi $\mathcal(X)^2(p)$.

kim tra, chng ta s t $p=20$ v xem chng ta nhn c g.

Fitting GARCH(1, 1) vi MLE

Mt khi chng ta quyt nh rng d liu c th c mt m hnh GARCH(1, 1) tim n, chng ta s tin hnh iu chnh GARCH(1, 1) cho d liu bng cch c lng cc tham s.

lm iu ny, chng ta cn hm log-likelihood

$$\mathcal{L}(\theta) = \sum_{t=1}^T - \ln \sqrt{2\pi} - \frac{x_t^2}{2\sigma_t^2} - \frac{1}{2}\ln(\sigma_t^2)$$

nh gi hm ny, chng ta cn $x_t$ v $\sigma_t$ cho $1 \leq t \leq T$. Chng ta c $x_t$, nhng chng ta cn tnh ton $\sigma_t$. lm iu ny, chng ta cn a ra mt c lng cho $\sigma_1$. D on ca chng ta s l $\sigma_1^2 = \hat E[x_t^2]$. Khi c c lng ban u, chng ta tnh ton cc gi tr cn li ca $\sigma$ bng cng thc

$$\sigma_t^2 = a_0 + a_1 x_{t-1}^2 + b_1\sigma_{t-1}^2$$

# Here's our function to compute the sigmas given the initial guess
def compute_squared_sigmas(X, initial_sigma, theta):

    a0 = theta[0]
    a1 = theta[1]
    b1 = theta[2]

    T = len(X)
    sigma2 = np.ndarray(T)

    sigma2[0] = initial_sigma ** 2

    for t in range(1, T):
        # Here's where we apply the equation
        sigma2[t] = a0 + a1 * X[t-1]**2 + b1 * sigma2[t-1]

    return sigma2

def negative_log_likelihood(X, theta):

    T = len(X)

    # Estimate initial sigma squared
    initial_sigma = np.sqrt(np.mean(X ** 2))

    # Generate the squared sigma values
    sigma2 = compute_squared_sigmas(X, initial_sigma, theta)

    # Now actually compute
    return -sum(
        [-np.log(np.sqrt(2.0 * np.pi)) -
        (X[t] ** 2) / (2.0 * sigma2[t]) -
        0.5 * np.log(sigma2[t]) for
         t in range(T)]
    )

By gi chng ta thc hin ti u ha s tm c lng ca chng ta cho $$\hat\theta = \arg \max_{(a_0, a_1, b_1)}\mathcal{L}(\theta) = \arg \min_{(a_0, a_1, b_1)}-\mathcal{L}(\theta)$$

Chng ta c mt s rng buc v iu ny

$$a_1 \geq 0, b_1 \geq 0, a_1+b_1 < 1$$

GMM c Lng Tham S GARCH(1, 1)

Chng ta va tnh ton mt c lng bng phng php MLE, nhng chng ta cng c th s dng Phng Php Tng Qut Thi im (GMM) c lng cc tham s GARCH(1, 1).

lm iu ny, chng ta cn nh ngha cc thi im ca mnh. Chng ta s s dng 4 thi im.

  1. Phn d $\hat\epsilon_t = x_t / \hat\sigma_t$
  2. Phng sai ca phn d $\hat\epsilon_t^2$
  3. Thi im lch $\mu_3/\hat\sigma_t^3 = (\hat\epsilon_t - E[\hat\epsilon_t])^3 / \hat\sigma_t^3$
  4. Thi im nhn $\mu_4/\hat\sigma_t^4 = (\hat\epsilon_t - E[\hat\epsilon_t])^4 / \hat\sigma_t^4$
In[15]:
# The n-th standardized moment
# skewness is 3, kurtosis is 4
def standardized_moment(x, mu, sigma, n):
    return ((x - mu) ** n) / (sigma ** n)

GMM hin c ba bc.

Bt u vi $W$ l ma trn n v.

  1. c lng $\hat\theta_1$ bng cch s dng ti u ha s ti thiu ha

$$\min_{\theta \in \Theta} \left(\frac{1}{T} \sum_{t=1}^T g(x_t, \hat\theta)\right)' W \left(\frac{1}{T}\sum_{t=1}^T g(x_t, \hat\theta)\right)$$

  1. Tnh li $W$ da trn phng sai ca cc c lng $\theta$. (Tp trung nhiu hn vo cc tham s c sc gii thch)

$$\hat W_{i+1} = \left(\frac{1}{T}\sum_{t=1}^T g(x_t, \hat\theta_i)g(x_t, \hat\theta_i)'\right)^{-1}$$

  1. Lp li cho n khi $|\hat\theta_{i+1} - \hat\theta_i| < \epsilon$ hoc chng ta t n ngng lp.

Khi to $W$ v $T$ v nh ngha hm mc tiu m chng ta cn ti thiu ha.

In[16]:
def gmm_objective(X, W, theta):
    # Compute the residuals for X and theta
    initial_sigma = np.sqrt(np.mean(X ** 2))
    sigma = np.sqrt(compute_squared_sigmas(X, initial_sigma, theta))
    e = X / sigma

    # Compute the mean moments
    m1 = np.mean(e)
    m2 = np.mean(e ** 2) - 1
    m3 = np.mean(standardized_moment(e, np.mean(e), np.std(e), 3))
    m4 = np.mean(standardized_moment(e, np.mean(e), np.std(e), 4) - 3)

    G = np.matrix([m1, m2, m3, m4]).T

    return (G.T * W * G).item()

def gmm_variance(X, theta):
    # Compute the residuals for X and theta
    initial_sigma = np.sqrt(np.mean(X ** 2))
    sigma = np.sqrt(compute_squared_sigmas(X, initial_sigma, theta))
    e = X / sigma

    # Compute the squared moments
    m1 = e ** 2
    m2 = (e ** 2 - 1) ** 2
    m3 = standardized_moment(e, np.mean(e), np.std(e), 3) ** 2
    m4 = (standardized_moment(e, np.mean(e), np.std(e), 4) - 3) ** 2

    # Compute the covariance matrix g * g'
    T = len(X)
    s = np.ndarray((4, 1))
    for t in range(T):
        G = np.matrix([m1[t], m2[t], m3[t], m4[t]]).T
        s = s + G * G.T

    return s / T
