S khng n nh ca m hnh hi quy

Phn tch hi quy cho php chng ta c lng cc h s trong mt hm m tng i lin quan n nhiu tp d liu. Chng ta gi thuyt mt dng c th cho hm ny v sau tm cc h s ph hp vi d liu, lm vic di gi nh rng nhng sai lch so vi m hnh c th c coi l nhiu.

Khi xy dng mt m hnh nh vy, chng ta chp nhn rng n khng th d on chnh xc bin ph thuc. Ti y, chng ta mun nh gi chnh xc ca m hnh khng phi bng cch n gii thch bin ph thuc tt nh th no, m bng cch n n nh nh th no (tc l, cc h s hi quy n nh ra sao) trn n d liu mu ca chng ta. Cui cng, nu mt m hnh thc s l mt s ph hp tt, n nn tch bit c hai na ngu nhin ca tp d liu m chng ta m hnh ha ring bit. Ngc li, chng ta khng th gi nh rng m hnh khng ch l mt sn phm ca mu d liu c th m chng ta chn, hoc rng n s d on cc im d liu mi.

Chng ta s s dng hi quy tuyn tnh y minh ha, nhng nhng cn nhc tng t p dng cho tt c cc m hnh hi quy. Di y, chng ta nh ngha mt hm wrapper cho hi quy tuyn tnh t statsmodels c th s dng sau ny.

import numpy as np
import pandas as pd
from statsmodels import regression, stats
import statsmodels.api as sm
import matplotlib.pyplot as plt
import scipy as sp

def linreg(X,Y):
    # Running the linear regression
    x = sm.add_constant(X) # Add a row of 1's so that our model has a constant term
    model = regression.linear_model.OLS(Y, x).fit()
    return model.params[0], model.params[1] # Return the coefficients of the linear model

Phn tch hi quy rt nhy cm vi cc im ngoi lai. i khi nhng im ngoi lai ny cha thng tin, trong trng hp , chng ta mun xem xt chng; tuy nhin, trong nhng trng hp nh trn, chng c th ch l nhiu ngu nhin. Mc d chng ta thng c nhiu im d liu hn so vi v d trn, nhng chng ta c th gp (v d) s dao ng theo tun hoc thng, iu ny sau c th thay i ng k cc h s hi quy.

Thay i ch (Regime changes)

Thay i ch (hoc t bin cu trc) l khi c s thay i trong qu trnh to ra d liu, dn n cc mu sau ny tun theo mt phn phi khc. Di y, chng ta c th thy rng c mt thay i ch cho VNINDEX vo gia nm 2022, v vic phn chia d liu ti dn n mt s ph hp tt hn nhiu (mu ) so vi mt hi quy trn ton b tp d liu (mu vng). Trong trng hp ny, m hnh hi quy ca chng ta s khng d on c cc im d liu trong tng lai v h thng c bn khng cn ging nh trong mu na. Thc t, phn tch hi quy gi nh rng cc sai s l khng tng quan v c phng sai khng i, iu ny thng khng ng nu c s thay i ch .

# Manually set the point where we think a structural break occurs
breakpoint = 470
xs = np.arange(len(prices))
xs2 = np.arange(breakpoint)
xs3 = np.arange(len(prices) - breakpoint)

# Perform linear regressions on the full data set, the data up to the breakpoint, and the data after
a, b = linreg(xs, prices.values)
a2, b2 = linreg(xs2, prices[:breakpoint].values)
a3, b3 = linreg(xs3, prices[breakpoint:].values)

Y_hat = pd.Series(xs * b + a, index=prices.index)
Y_hat2 = pd.Series(xs2 * b2 + a2, index=prices.index[:breakpoint])
Y_hat3 = pd.Series(xs3 * b3 + a3, index=prices.index[breakpoint:])

# Plot the raw data
prices.plot()
Y_hat.plot(color='y')
Y_hat2.plot(color='r')
Y_hat3.plot(color='r')
plt.title('VNINDEX')
plt.ylabel('Price');

a cng tuyn

Trn y, chng ta ch xem xt hi quy ca mt bin ph thuc so vi mt bin c lp. Tuy nhin, chng ta cng c th c nhiu bin c lp. iu ny dn n s khng n nh nu cc bin c lp c mi tng quan cao.

Hy tng tng chng ta ang s dng hai bin c lp, $X_1$ v $X_2$, m c tng quan rt cao vi nhau. Khi , cc h s c th thay i mt cch ng k nu chng ta thm mt quan st mi m c gii thch tt hn bi mt trong hai bin hn l bin kia. Trong trng hp cc oan, nu $X_1 = X_2$, th s la chn cc h s s ph thuc vo thut ton hi quy tuyn tnh c th.

Di y, chng ta thc hin mt hi quy tuyn tnh a bin trong cc bin c lp c mi tng quan cao. Nu chng ta ly khong thi gian mu t 2023-01-01 n 2025-01-01, th cc h s xp x l 3.34 v 2.67. Nhng nu chng ta m rng khong thi gian n 2025-04-01, cc h s tr thnh xp x 3.63 v 2.9.
