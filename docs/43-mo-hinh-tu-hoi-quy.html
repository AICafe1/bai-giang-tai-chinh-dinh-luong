M hnh AR

M hnh t hi quy, hay AR$(p)$, c to ra bng cch hi quy mt chui thi gian trn cc gi tr trong qu kh ca n, tc l cc tr ca n. Dng n gin nht ca mt m hnh t hi quy l m hnh AR$(1)$, c ngha l ch s dng mt hng mc tr. Mt m hnh t tng quan bc nht nh vy cho mt chui thi gian $x_t$ l:

$$ x_t = b_0 + b_1 x_{t - 1} + \epsilon_t $$

Trong $x_{t - 1}$ i din cho gi tr ca chui thi gian ti thi im $(t - 1)$ v $\epsilon_t$ l hng mc sai s. Chng ta c th m rng iu ny thnh m hnh AR$(p)$, c k hiu:

$$ x_t = b_0 + b_1 x_{t-1} + b_2 x_{t - 2} \ldots + b_p x_{t - p} + \epsilon_t $$

mt m hnh AR hot ng ng, chng ta phi yu cu rng chui thi gian l n nh v hip phng sai. iu ny c ngha l n tun theo ba iu kin sau:

  1. Gi tr k vng ca chui thi gian l khng i v hu hn ti mi thi im, tc l $E[y_t] = \mu$ v $\mu < \infty$ cho tt c cc gi tr ca $t$.
  2. Phng sai ca chui thi gian l khng i v hu hn cho tt c cc khong thi gian.
  3. Hip phng sai ca chui thi gian vi chnh n trong mt s lng c nh cc khong thi gian trong tng lai hoc qu kh l khng i v hu hn cho tt c cc khong thi gian, tc l

$$ COV(y_t, y_{t - s}) = \lambda, \ |\lambda| < \infty, \text{ $\lambda$ khng i}, \ t = 1, 2, \ \ldots, T; \ s = 0, \pm 1, \pm 2, \ldots, \pm T $$

Lu rng biu din ton hc ny bao gm iu kin 2.

Nu cc iu kin ny khng c tha mn, kt qu c lng ca chng ta s khng c ngha thc t. Cc c lng cho cc tham s s b thin lch, lm cho bt k th nghim no m chng ta c gng thc hin bng cch s dng m hnh tr nn khng hp l. Tht khng may, vic tm mt chui thi gian n nh v hip phng sai trong thc t th trng ti chnh c th l mt thch thc. V d, khi chng ta xem xt gi c phiu ca FPT, chng ta c th thy r mt xu hng tng. Gi tr trung bnh ang tng theo thi gian. Tuy nhin, c nhng cch lm cho mt chui thi gian khng n nh tr nn n nh. Khi chng ta thc hin bin i ny, chng ta c th xy dng cc m hnh t hi quy di cc gi nh trn.

import numpy as np
from scipy import stats
from statsmodels.tsa.ar_model import AutoReg
import matplotlib.pyplot as plt

# ensures experiment runs the same every time
np.random.seed(100)

# This function simluates an AR process, generating a new value based on historial values,
# autoregressive coefficients b1 ... bk, and some randomness.
def AR(b, X, mu, sigma):
    l = min(len(b)-1, len(X))
    b0 = b[0]

    return b0 + np.dot(b[1:l+1], X[-l:]) + np.random.normal(mu, sigma)

b = np.array([0, 0.8, 0.1, 0.05])
X = np.array([1])

mu = 0
sigma = 1

for i in range(10000):
    X = np.append(X, AR(b, X, mu, sigma))

plt.plot(X)
plt.xlabel('Time')
plt.ylabel('AR Series Value');

Ri ro ui (Tail Risk)

Cc qu trnh t hi quy c xu hng c nhiu gi tr cc oan hn so vi d liu c ly t mt phn phi nh phn phi chun chng hn. iu ny l do gi tr ti mi im thi gian b nh hng bi cc gi tr gn y. Nu chui ngu nhin ny tng ln, n c kh nng s duy tr mc cao hn so vi mt chui khng phi t hi quy. iu ny c gi l 'tnh dy ui' (fat-tailedness) v cc gi tr cc oan trn hm mt xc sut (pdf) s dy hn so vi trong mt phn phi chun.

Nhiu cuc tho lun v ri ro ui trong ti chnh xut pht t thc t rng cc s kin ui thc s xy ra v kh m hnh ha do tn sut xy ra ca chng thp. Nu chng ta c l do nghi ng rng mt qu trnh l t hi quy, chng ta nn mong i ri ro t cc s kin cc oan v iu chnh cho ph hp.

M hnh AR ch l mt trong nhng ngun gc ca ri ro ui, v vy ng cho rng v mt chui khng phi l AR, n khng c ri ro ui.

def compare_tails_to_normal(X):
    # Define matrix to store comparisons
    A = np.zeros((2,4))
    for k in range(4):
        #stores tail probabilities of the sample series vs a normal series
        A[0, k] = len(X[X > (k + 1)]) / float(len(X)) # Estimate tails of X
        A[1, k] = 1 - stats.norm.cdf(k + 1) # Compare to Gaussian distribution
    print('Frequency of std events in X \n1: %s\t2: %s\t3: %s\t4: %s' % tuple(A[0]))
    print('Frequency of std events in a normal process \n1: %s\t2: %s\t3: %s\t4: %s' % tuple(A[1]))
    return A

def compute_unadjusted_interval(X):
    T = len(X)
    # Compute mu and sigma MLE
    mu = np.mean(X)
    sigma = np.std(X)
    # Compute the bounds using standard error
    lower = mu - 1.96 * (sigma/np.sqrt(T))
    upper = mu + 1.96 * (sigma/np.sqrt(T))
    return lower, upper

# We'll make a function that returns true when the computed bounds contain 0
def check_unadjusted_coverage(X):
    l, u = compute_unadjusted_interval(X)
    # Check to make sure l <= 0 <= u
    if l <= 0 and u >= 0:
        return True
    else:
        return False

def simululate_AR_process(b, T):
    X = np.array([1])

    mu = 0
    sigma = 1

    for i in range(T):
        X = np.append(X, AR(b, X, mu, sigma))

    return X

Kim tra Hnh vi AR

xc nh bc, $p$, ca m hnh AR$(p)$, chng ta xem xt cc t tng quan ca chui thi gian. y l cc tng quan ca chui vi cc gi tr trong qu kh ca n. T tng quan bc $k$ c nh ngha l

$$ \rho_k = \frac{COV(x_t, x_{t - k})}{\sigma_x^2} = \frac{E[(x_t - \mu)(x_{t - k} - \mu)]}{\sigma_x^2} $$

Trong $k$ i din cho s khong thi gian tr. Chng ta khng th quan st trc tip cc t tng quan, v vy chng ta c lng chng nh sau:

$$ \hat{\rho}_k = \frac{\sum_{t = k + 1}^T[(x_t - \bar{x})(x_{t - k} - \bar{x})]}{\sum_{t = 1}^T (x_t - \bar{x})^2} $$

phc v cho mc ch ca mnh, chng ta c th s dng mt cp cng c gi l hm t tng quan (ACF) v hm t tng quan ring (PACF) xc nh bc ca m hnh. PACF kim sot cho cc tr ngn hn, khc vi ACF. Cc hm ny c bao gm trong nhiu gi phn mm thng k v tnh ton cc t tng quan mu cho chng ta, cho php chng ta xc nh gi tr ph hp ca $p$.

def plot_acf(X_acf, X_acf_confs, title='ACF'):
    # The confidence intervals are returned by the functions as (lower, upper)
    # The plotting function needs them in the form (x-lower, upper-x)
    errorbars = np.ndarray((2, len(X_acf)))
    errorbars[0, :] = X_acf - X_acf_confs[:,0]
    errorbars[1, :] = X_acf_confs[:,1] - X_acf

    plt.plot(X_acf, 'ro')
    plt.errorbar(range(len(X_acf)), X_acf, yerr=errorbars, fmt='none', ecolor='gray', capthick=2)
    plt.xlabel('Lag')
    plt.ylabel('Autocorrelation')
    plt.title(title);

Cc c Tnh S a Ra Qu Nhiu Tr

Chng ta c th thy m hnh ca chng ta c tnh kh nhiu tham s. Trong trng hp ny, chng ta bit c qu nhiu tham s v chng ta m phng d liu nh mt qu trnh AR(3). L do m cc m hnh AR s c tnh nhiu tr hn so vi thc t l do s ph thuc gin tip. Nu $X_t$ ph thuc vo $X_{t-1}$, th mt cch gin tip v mc thp hn, n s ph thuc vo $X_{t-2}$. Trong s hin din ca nhiu hn mt tr trong qu trnh sinh d liu, chng ta s c nhng cu trc hi ho phc tp trong cc tr. Nhng s ph thuc gin tip ny s c mt c tnh n gin pht hin.

Bn Mun S Tham S t Nht C Mt M Hnh Tt

Ni chung, him khi bn c th thu c iu g hu ch t mt m hnh vi nhiu tham s, xem bi ging v Overfitting bit l do. Trong trng hp ny, chng ta mun chn mt s lng tr m chng ta tin l gii thch c nhng g ang din ra, nhng khng b overfitting v chn mt m hnh vi qu nhiu tr.

Quan st ACF v PACF cho thy ch c 3 tr u tin c th hu ch. Tuy nhin, chng ti s m rng s lng tr ln 10 kim tra li d liu ban u ca chng ti. Chng ti s s dng tiu ch thng tin, c th l Tiu ch Thng tin Akaike (AIC) v Tiu ch Thng tin Bayes (BIC) quyt nh s lng tham s chnh xc.

Vic gii thch AIC v BIC c thc hin nh sau. Tnh ton AIC v BIC cho tt c cc m hnh m chng ta mun xem xt, v ghi li AIC v BIC nh nht c ghi li l $AIC_{min}$ v $BIC_{min}$. y l cc m hnh m ti thiu ha mt mt thng tin theo tng tiu ch. i vi mi loi IC, chng ta c th tnh ton kh nng tng i ca mi m hnh $i$ bng cch ly

$$l = e^{(IC_{min} - IC_{i})/2}$$

Chng ta c th hiu $l$ nh l m hnh $i$ c kh nng $l$ ln ti thiu ha mt mt thng tin, so vi m hnh AIC ti thiu. C th cn c li mt vi ln hiu iu ny, v vy hy cng xem n hot ng nh th no.
